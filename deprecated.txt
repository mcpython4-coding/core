This file is about things that should be removed in the future.
This is an reminder-file for when to remove them & what.
So, here follows an list of deprecation's:

since 23th of november, 2019:
    Language.get(key, formatting=None):
        uses an key-splitting method. This should be removed and replaced by formatting.
        replace all internal calls from state lables, ... when removing this behaviour

        removal: until next ~2 snapshot

    Language.decode(s: str):
        use function translate rather than this function

        removal: until next ~2 snapshot

since 24th of november, 2019:
    ResourceLocator.RESOURCE_PACK_LOADERS, former ResourceLocator.RESOURCE_LOADER
        removal: until next ~2 snapshot

    ResourceLoader.load_resources()
        use load_resource_packs instead, removal: until next ~2 snapshot

    setup.IPrepareAbleTask.uses_directory()
        use variable USES_DIRECTORY instead, removal: until next ~2 snapshot

        todo: deprecate get_name of the task

    block.Block.Block.on_create()
        use __init__ with super call instead
        removal: until next ~2 snapshot

    block.Block.Block.on_delete()
        use on_remove instead
        removal: until next ~2 snapshot
