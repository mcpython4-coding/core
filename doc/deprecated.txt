This file is about things that should be removed in the future.
This is an reminder-file for when to remove them & what.
So, here follows an list of deprecation's:

general strategy for feature moving:
    1. mark for removal in the future
    2. add new method linking to old, deprecate old
    3. link old to new
    4. remove old, add line stating: # function <XY> removed in <version>
    5. remove this
